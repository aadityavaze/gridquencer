Gridquencer
===========

A sequencer for grid interfaces, starting with the Ableton Push. 

Installation
------------

Checkout repo

	> git clone git@github.com:drart/gridquencer.git

Install node.js modules:

	> npm install 

Set up IAC Bus 1
----------------

https://support.apple.com/en-ca/guide/audio-midi-setup/ams1013/mac

Running
-------

    > node gridquencer.js
